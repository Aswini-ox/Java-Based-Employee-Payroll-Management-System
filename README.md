# Employee Payroll Management System (Java)

## Project Description
This is a console-based Employee Payroll Management System developed using Java.
It allows users to add employee details and generate a salary pay slip.

## Features
- Add employee details
- Calculate net salary
- Display employee pay slip
- Menu-driven system
- Attractive console output

## Technologies Used
- Java
- VS Code

## How to Run
1. Compile the program:
   javac PayrollManagement.java
2. Run the program:
   java PayrollManagement
